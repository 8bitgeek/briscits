# BRISCITS
Basic RISC Intrinsic Time Scheduler = Lightweight preemtive multitasking for RISC micro-controllers

# Supported Chips

* GD32VF103xB

# BSP Support

* seeedstudio-gd32
* sipeed-longan-nano

# Make Examples

* make -f bsp/seeedstudio-gd32/Makefile 
* make -f bsp/sipeed-longan-nano/Makefile
